### 能力模式七: Skill调用 + 多模态内容自动生产 (支撑一+三组合) 📱🖥️ 新增

> **模式说明**: 任务模式定时或条件触发已安装的Skill（如doubao-creative-video），自动完成从文字/图片信息到成品视频/内容的端到端生产。区别于能力模式五的"多步链条手动组合工具"——本模式的核心是"Skill内部封装了完整生产策略，任务模式只负责调度+上下文持久化"。

**案例16 · 餐饮店主每日菜单口播短视频自动生成** ✅已实测分析（UGC路线，doubao-creative-video Skill）
- 人群：餐饮店主、面包房、熟食店、小吃摊（全国约900万餐饮商户）
- 痛点：每天需要发短视频引流，但录一条口播要备货、打光、念稿、重录三四遍。不用真人出镜的图文流量又差。录两天就放弃了
- 流程：每天上午 → 店主输入今日主推菜品+卖点 → 任务模式调度doubao-creative-video Skill → 走UGC路线生成15秒口播短视频（自动匹配主播模板、产品一致性锁定、安利脚本去AI腔）→ 推手机
- 传播点："别人还在想今天拍什么，AI已经把口播视频录好了——不嘴瓢、不怯场、比你自己录的还稳"
- 模式亮点：Skill内部封装了完整UGC生产策略（主播抽卡+产品锁+安利脚本引擎+去AI腔规则+分镜硬门控）。任务模式的价值是：记住店铺品牌调性、跨日保持主播不撞脸、每次只输入菜品名即可触发完整链路
- Skill路由：doubao-creative-video → ugc-talking-video-ref.md
- 关键验证点：脚本去AI腔质量、产品一致性锁在无参考图时的表现、跨日主播去重

**案例17 · 实体商家节日营销视频自动排期生产** ✅已出测试提示词（Marketing路线，doubao-creative-video Skill）
- 人群：实体零售店主、社区服务商、餐饮老板（全国约8000万个体户）
- 痛点：节气/节日要做营销内容但记不住节点；想起来只剩一天来不及做；临时拼凑的图+文字效果=发了等于没发
- 流程：设置行业和店铺类型 → 每个节日/节气前5天任务模式自动提醒 → 用户确认营销诉求 → 调度doubao-creative-video走Marketing路线生成15秒促销短视频 → 推手机
- 传播点："别人的店还在群里问'端午发什么'，AI提前一周把营销视频做好了"
- Skill路由：doubao-creative-video → product-marketing-ad-video-no-storyboard-ref.md

**案例18 · 小微企业主企业宣传视频自动生成** ✅已出测试提示词（Corporate路线，doubao-creative-video Skill）
- 人群：小微企业主、个体创业者（全国1.7亿市场主体）
- 痛点：找制作公司一个宣传片5-20万；自己拍不知道说什么。连一个像样的企业介绍视频都拿不出来
- 流程：上传logo+业务介绍 → 任务模式调度doubao-creative-video走Corporate路线 → 自动搜索公开信息 → 生成15秒企业宣传短视频（叙事母题+证据链+品牌资产锁定）→ 推手机
- 传播点："别人花5万做企业宣传片，AI一晚上帮你生成好了"
- Skill路由：doubao-creative-video → corporate-business-video-ref.md
- 注意：此案例测试复杂度最高（四类必填信息+公开搜索+事实核查），建议在案例16/17跑通后再测
