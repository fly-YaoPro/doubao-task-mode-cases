# 案例十五：财经自媒体作者·每日开盘前市场热点简报

> 拆分自「金融方向案例」（海外版豆包案例的本地化测试集），Skill 路由：doubao-market-hotspot

### 过筛

| 筛 | 判断 |
|----|------|
| 第0筛 | ✅ 财经内容创作——生产力/经营活动 |
| 第一筛 | 对话能做一次热点解读，但每天早上手动检索+梳理+写稿=不可能每天坚持 |
| 第二筛 | 无App能记住内容风格偏好+每天自动跑热点分析+输出固定结构 |
| 第三筛 | ✅ 财经自媒体（公众号/雪球/抖音）从业者数十万，每天需要"今天写什么"的选题和素材 |

**对话模式 vs 任务模式**：对话=每天早上手动搜新闻→手动整理→手动写稿。任务模式=早8:00自动出热点简报→人只需要确认和改写。

### 造假背景

**人物**：小刀，前财经记者，现在做自己的财经公众号"刀哥看市"，5万粉丝。每天发一篇市场解读，风格是"说人话、讲逻辑、不装神弄鬼"。

**困境**：每天早上醒来第一件事就是翻华尔街见闻、财联社、金十数据——看昨晚外盘怎么样、今天A股有什么大事、政策面有什么新变化。然后花1-2小时梳理成一篇1500字的文章。这件事每天早上都要做，雷打不动。最痛苦的是周末想休息一天——结果周一早上要补周末两天的信息量，直接崩溃。

**造假数据·内容风格偏好**：

```
账号：刀哥看市
定位：A股市场每日解读，说人话版本
风格：口语化、有态度、逻辑链条清晰、不堆数据
固定结构：
  1. 一句话核心判断（今天市场最值得关注的一件事）
  2. 昨天发生了什么（A股+外盘+商品三句话）
  3. 今天盯什么（政策/事件/数据）
  4. 刀哥怎么看（300字观点，有论据有边界）
字数：1200-1500字
发布时间：每天早上8:30
```

### 场景测试

**预判路由**：doubao-market-hotspot → 对话框正文（7段叙事链）

**为什么匹配**：skill明确支持"市场整体与宏观事件解读"，输出结构恰好是7段叙事链——核心观点→增量信息→为什么重要→如何影响市场→各方观点→验证/证伪信号→信息来源。这个结构可以直接作为公众号文章的素材骨架。

**提示词**：

```
帮我创建每日开盘前市场热点自动简报：

【路由】doubao-market-hotspot → 对话框正文

【持久化配置】
- 市场范围：A股为主+隔夜外盘（美股/港股）+重要大宗商品（原油/铜/黄金）
- 时间窗：近24小时（交易日覆盖昨日收盘到今早开盘前的所有增量）
- 分析模式：全市场默认模式

【内容风格模板】（每次复用）
- 账号：刀哥看市
- 定位：说人话的市场解读
- 风格：口语化但不轻浮、有态度但讲逻辑
- 读者画像：有基本投资常识的散户，不需要解释什么叫PE什么叫MACD
- 固定结构（对应skill的7段叙事链）：

  【核心判断】（对应"核心观点"）
  用一句话说今天最值得关注的一件事。例如：
  "昨晚美联储放鸽，今天A股大概率高开。但别高兴太早——高开之后能不能撑住，取决于北向资金是不是真的进场。"

  【隔夜发生了什么】（对应"有哪些关键增量信息"）
  三句话讲清：A股昨收+外盘隔夜+大宗早盘。只写数字和事实，不加评论。

  【这些事为什么重要】（对应"这些信息为什么重要"+ "如何影响市场"）
  解释增量信息背后的真正变化和预期差。例如：
  "美联储这次不是'鸽'了利率，而是'鸽'了对通胀的表述——从'仍然偏高'改成了'取得进一步进展'。这意味着9月降息概率从60%跳到75%。"

  【今天盯什么】（对应"验证/证伪信号"）
  今天盘中需要盯的3个信号。例如：
  "盯北向资金开盘15分钟的净流向（如果净流入超20亿→外资在真金白银买）；盯A股券商板块（券商先动=情绪真的回暖）"

  【刀哥怎么看】（对应"核心观点"的展开）
  300字观点。有论据、有边界、有"如果XX则XX"的条件推演。不写"明天必涨"这种垃圾话。

- 字数：1200-1500字
- 不要：emoji、过度排比、AI套话（"值得关注的是""综上所述""毋庸置疑"）
- 要：具体数字、具体日期、具体信号、条件推演（"如果XX则XX，否则XX"）

【触发】每个交易日早上8:00

【执行流程】
1. 调用doubao-market-hotspot，获取昨日收盘到今早的增量信息
2. 按上述内容风格模板，将skill的7段叙事链改写为"刀哥看市"的口语化版本
3. 推送飞书"刀哥·内容素材"：标题建议+完整文章素材
4. 同时保存到"E:\刀哥看市\市场简报\{日期}.md"

【需要记住的】
- 昨天简报中的核心判断——今早如果被验证/证伪了，要在开头提一句
- 上周提到过的"待观察信号"——如果本周出现了，要点出来

【异常处理】
- 如果隔夜外盘数据不可得→标注"外盘数据暂未更新"
- 如果当日无增量信息（极其罕见的新闻真空）→推送"今日无重大增量，延续昨日判断"
- 非交易日→跳过
```

**测试关注点**：

| 关注项 | 描述 | 风险 |
|--------|------|------|
| 事实准确性 | skill要求"关键事实必须有可打开来源"，简报中的数字/日期/事件是否可追溯 | 🔴 |
| 口语化改写 | 从skill的7段正式叙事→"刀哥"口语风格的转化是否自然 | 🟡 |
| 跨日逻辑连贯 | 昨天的判断今天是否被追踪（验证/证伪） | 🟡 |
| 字数控制 | 命中的1200-1500字是否稳定（不150字敷衍也不3000字啰嗦） | 🟢 |
| 合规 | 有无"必涨""肯定""绝对"等违规词 | 🔴 |
