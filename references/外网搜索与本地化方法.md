# 外网搜索与本地化方法

## 案例来源：外网搜索与本地化方法

### 为什么需要这一步
Codex 和 Codex 在外网有大量真实用户分享的实战案例（Reddit、Hacker News、YouTube、官方论坛等）。这些案例经过了真实用户验证，可以用来发现新能力模式，然后本地化成国内可落地的豆包案例。**特别关注外网用户分享的Skill/workflow使用场景——这些可以直接对飙豆包的Skill生态。**

### 搜索源与搜索策略

**外网优质源**:
- Reddit: r/ClaudeAI, r/Codex, r/singularity, r/OpenAI
  - 搜索词: "Codex use case", "automated", "scheduled", "monitoring", "daily task", "skill workflow", "custom slash command"
  - 精华帖搜索: sort by top/comments, filter by year
  - 优先看高赞帖子的评论区——真正的使用场景往往在评论里
- Hacker News: hn.algolia.com 搜索 "Codex" + "automation" / "scheduled" / "use case"
- Anthropic 官方: Codex 文档中的 Cookbook / Examples 栏目
- YouTube: "Codex workflow", "Codex automation" 视频描述中的使用场景
- Codex 论坛/Showcase: OpenAI 官方论坛中 Codex 用户分享的实战案例
- **新增**: GitHub Awesome Lists: "awesome-Codex", "awesome-codex" —— 社区整理的skill/workflow清单

**国内优质源**:
- 即刻、V2EX、少数派: 搜索 "Codex" + "自动化" / "每天" / "定时" / "监控"
- 小红书: 搜索 "Codex 怎么用" / "AI 帮我干活" / "豆包 skill"
- 知乎: 搜索 "Codex 实际应用" / "AI 工作流"
- **新增**: 飞书/钉钉开发者社区 —— 国内企业用户分享的自动化场景

**搜索关键词组合**:
- 功能型: "night shift", "babysitting", "scheduled task", "morning briefing", "weekly report", "skill workflow", "agent automation"
- 人群型: "small business", "restaurant", "freelancer", "landlord", "logistics"
- 行为型: "monitoring", "checking", "scraping", "comparing", "reminding", "generating reports", "building dashboards"

### 本地化方法

**三步本地化流程**:
1. 读原帖 → 提取核心能力模式（不是照搬场景，而是提取"能力动作"）
2. 找国内对标 → 相同的"能力动作"在国内有什么人群/场景需要？
3. 过五筛 → 确认成立后改写为豆包案例格式

**本地化示例（原有）**:
- 外网: "Codex monitors my Shopify competitors' pricing and pings me on Slack"
- 提取能力: 公开商业页面价格监控 + 条件触发推送
- 国内对标: 淘宝/拼多多/抖音小店竞品公开商详页 → 案例 K1 竞品定价哨兵

- 外网: "Codex runs my morning market briefing: scrapes Fed statements, compares wording changes"
- 提取能力: 公开政策文档定时抓取 + 新旧对比 + 推送
- 国内对标: 商务部/海关总署/税务总局关税公告 → 案例 K2 外贸关税盯梢

**本地化示例（新增Skill维度）**:
- 外网: "I use Codex with a custom workflow to generate product demo videos from my Shopify images every week"
- 提取能力: Skill调度 + 定时触发 + 产品图→视频自动生产
- 国内对标: doubao-creative-video Skill + 任务模式定时 → 案例16/17/18

- 外网: "Codex builds me a weekly sales dashboard that auto-updates from my database"
- 提取能力: Skill调度 + 定时 + 数据→看板自动生成
- 国内对标: doubao-app-builder Skill + 任务模式定时 → 案例19

**常见本地化陷阱**:
- Slack/Discord 推送 → 国内是微信/飞书/钉钉/短信推送
- Shopify/Etsy 卖家 → 淘宝/拼多多/抖音小店卖家
- IRS/SEC/FDA → 税务局/证监会/药监局
- 美股/加密货币 → A股/ 国内合规投资标的

### 结合八个能力模式搜索
- 模式一~六: 使用原有搜索词
- 模式七(Skill+多模态): 搜 "AI video generation workflow", "auto-generate product videos", "scheduled content creation", "Codex video skill"
- 模式八(Skill+应用生成): 搜 "AI dashboard builder", "auto-generate reports", "Codex app builder", "no-code app generation", "weekly report automation"
