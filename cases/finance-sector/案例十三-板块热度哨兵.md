# 案例十三：私募研究员·每日自选板块开盘前热度哨兵

> 拆分自「金融方向案例」（海外版豆包案例的本地化测试集），Skill 路由：doubao-finance-sector

### 过筛

| 筛 | 判断 |
|----|------|
| 第0筛 | ✅ 投资研究——生产力/职业场景 |
| 第一筛 | 对话能做一次板块分析，但每天早上手动查3个板块=3次对话×手动对比=无法坚持 |
| 第二筛 | 无App能记住自选板块列表+每天自动跑分析+自动判断背离信号 |
| 第三筛 | ✅ A股散户2亿+，每天看板块热度是开盘前刚需 |
| PC-1 | ✅ 投资研究天然在电脑前（看盘软件+分析报告） |
| PC-2 | ✅ 3板块×5维度数据=高信息密度 |
| PC-3 | ✅ 需在行情软件/浏览器/报告之间切换 |

**对话模式 vs 任务模式**：对话=每天早上手动输入"帮我分析AI板块热度"×3次→手动对比→手动判断。任务模式=一次配置自选板块→每天8:30自动跑3个板块→推对比摘要+背离信号→人只需要看推送决定今天做不做。

### 造假背景

**人物**：老徐，38岁，前券商研究员，现在自己做二级市场投资。管理自有资金约500万，风格是中短线板块轮动——哪个板块开始热了就去哪个板块做一波。

**困境**：每天早上要做三件事——打开同花顺看板块涨幅榜→翻财经新闻看有没有新催化→判断3-5个关注的板块今天能不能做。这个过程每天早上耗40分钟，而且经常是开盘了还没看完。最烦的是——有时候板块悄悄热了两天了（比如低空经济连续小阳线+成交温和放大），他没注意到，等新闻出来的时候已经涨了10%，追进去就站岗。

他最需要的是：每天早上8:30，有人告诉他——"你关注的3个板块里，今天AI应用概念信息热度在上升但行情还没动（背离信号→未启动），低空经济已经在放量加速（强趋势共振），消费电子没动静（低关注）。今天优先看AI应用概念。"

**造假数据·自选板块列表**：

```
板块1：AI应用概念（AI agent / AI软件 / 大模型应用）
板块2：低空经济（eVTOL / 无人机 / 低空基建）
板块3：消费电子（手机链 / 果链 / 华为链）

关注逻辑：
- AI应用：政策密集+海外映射强，但国内标的业绩兑现度低→波动大，适合提前埋伏
- 低空经济：今年主线之一，但有政策落地节奏问题→需要盯政策催化节点
- 消费电子：周期反转逻辑，但需要盯月度出货数据和库存水位
```

### 场景测试

**预判路由**：doubao-finance-sector → 第一阶段对话框分析

**为什么匹配**：skill明确支持"判断板块短期热度/持续性/能不能追"。三板块哨兵=三次独立的skill调用，任务模式负责定时触发+结果对比+推送。

**提示词**：

```
帮我创建每日开盘前板块热度哨兵：

【路由】doubao-finance-sector（对每个板块独立调用第一阶段分析）

【持久化自选板块】
板块1：AI应用概念（AI agent/大模型应用/AI软件）
- 关注原因：政策密集+海外映射强+高波动→适合提前埋伏
- 重点关注信号：海外AI公司财报超预期、国内AI应用政策出台、龙头股突破前高

板块2：低空经济（eVTOL/无人机/低空基建）
- 关注原因：今年主线+政策驱动→需要盯催化节点
- 重点关注信号：适航证进展、地方低空经济政策、龙头股放量突破

板块3：消费电子（手机链/果链/华为链）
- 关注原因：周期反转逻辑→需要盯月度数据
- 重点关注信号：月度出货数据、库存水位变化、苹果/华为新品供应链消息

【触发】每个交易日早上8:30（开盘前30分钟）

【执行流程】
Step 1 · 对每个板块独立调用doubao-finance-sector第一阶段分析
→ 每个板块获取：
  - 核心结论（现在热不热、能不能追）
  - 双轨打分（信息热度1-5 + 行情热度1-5 + 综合分）
  - 背离信号（未启动/纯资金驱动/强趋势共振/低关注）
  - 今日核心催化（3-5条）
  - 10只代表股的强弱分化
  - 接下来盯什么信号

Step 2 · 生成三板块对比摘要
按以下格式推送飞书"老徐·投资研究"：

```
📊 {日期} 开盘前板块热度哨兵

🔴 AI应用概念 | 综合3.1 | 信息↑行情→ | 背离：未启动
催化：OpenAI发布新Agent API / 国内XX政策明日出台
10股代表：XX涨停、XX放量突破、XX缩量震荡……
信号：关注龙头能否站稳前高，如果明日继续放量→进入强趋势

🟢 低空经济 | 综合4.2 | 信息↑行情↑ | 背离：强趋势共振
催化：亿航OC证获批预期升温 / 深圳低空基建招标
10股代表：XX连续3日放量、XX创年内新高……
信号：连续放量但未加速→短期健康。关注亿航OC证落地时间

⚪ 消费电子 | 综合2.0 | 信息→行情→ | 背离：低关注
催化：无新增催化。6月手机出货数据本周五公布
10股代表：全线缩量震荡，无个股异动……
信号：等周五出货数据。如果超预期→可能触发补涨
```

Step 3 · 推送后的交互
- 如果某个板块出现"背离：未启动"信号→推送中加一条额外提示："AI应用概念信息热度上升但行情未启动——如果你感兴趣，回复'AI'我帮你生成完整仪表盘"
- 如果用户回复板块名→触发该板块第二阶段（生成HTML仪表盘）

【需要记住的】
- 自选板块列表+关注逻辑（可随时增删改）
- 每个板块历史热度变化趋势（便于跨日对比：今天是热了还是凉了）
- 上次出现"背离：未启动"信号的板块和日期（用于跟踪信号是否兑现）

【异常处理】
- 如果某个板块当日无法获取足够数据→推送中标注"数据不足，跳过"
- 如果seed_finance_search返回异常→降级到general_search做简化分析
- 如果非交易日（周六日/节假日）→自动跳过，不推送
```

**测试关注点**：

| 关注项 | 描述 | 风险 |
|--------|------|------|
| 双轨打分一致性 | 三个板块的打分标准是否统一（不能用不同尺度） | 🟡 |
| 背离检测准确性 | "未启动/纯资金/强趋势/低关注"是否正确判断 | 🔴 |
| 10股代表取数 | 每个板块的代表股是否真的代表了该板块 | 🟡 |
| 数据来源可追溯 | 行情数据/催化来源是否可打开核实（skill硬要求） | 🔴 |
| 跨日趋势 | 能不能和昨天的热度对比（热了还是凉了） | 🟡 |
| 定时触发准确性 | 交易日早8:30是否准时触发 | 🟡 |
| 推送格式可读性 | 三板块对比在一屏内是否清晰 | 🟢 |

### 场景发散

- **公募基金经理**：盯10个行业指数→每天早上出行业轮动哨兵
- **期货交易员**：盯3-5个商品板块→每天早上出商品热度哨兵
- **美股投资者**：盯纳斯达克几个赛道→每天北京时间晚上开盘前出哨兵（时区适配）
- **财经自媒体**：每天早上出"今日最热板块Top3"→直接变成公众号素材
