# 案例七（深度版）：跨境电商·多语种产品视频独立生成

> 修订：2026-06-30 | 原修正版→加上造假素材+完整三语种脚本
> 预判路由：doubao-creative-video → UGC路线 → ugc-talking-video-ref.md


## 过筛

| 筛 | 判断 |
|----|------|
| 第0筛 | ✅ 跨境电商经营活动 |
| 第一筛 | 对话能做一条多语种视频，但每周处理多SKU×3语种=手动输入6-9次brief，记不住跨语种的产品一致性参数 |
| 第三筛 | ✅ 跨境电商卖家核心痛点——多语种内容成本占营销预算30%+ |
| PC-1 | ✅ 视频文件管理+分发在电脑端 |

**对话模式 vs 任务模式**：对话=每条视频手动写brief×3语种×N个产品×每周。任务模式=一次配置产品信息→每周自动为新产品生成三个语种版本，产品外观基于参考图锁定，品牌名/参数永不变。


## 造假背景

**公司**："赛音科技"（Saiyin，虚构），深圳宝安一家蓝牙耳机工厂。2018年开始做OEM/ODM，给海外品牌供货。2026年注册自有品牌"Saiyin"，第一款产品T10真无线降噪耳机上市。

**人物**：
- 老周（老板）：管工厂和供应链，不懂海外营销
- 小杨（唯一的海外运营）：一个人管Shopee印尼站、Lazada泰国站、Amazon美国站三个渠道

**困境**：小杨用中文版抖音带货视频加了英文字幕，同时投印尼和泰国。结果——印尼退货率15%（看不懂中英文字幕，纯粹被画面吸引就买了，拿到手不会连蓝牙），泰国站转化率只有国内的1/3（没有本地化内容）。

问题是：一条视频翻成印尼语+泰语+英语，找翻译+找配音+重剪字幕=一条本地化成本比原片还贵。不本地化就没转化，本地化就亏钱。

**任务模式破局**：小杨一次配置好T10的产品信息（产品图+核心卖点+品牌参数）→任务模式记住这些→每周自动产出三个语种版本的独立UGC种草视频。不是"一个视频翻三个语种"，而是"同一个产品生成三条不同风格、不同主播、不同场景的当地化视频"——成本从"找三个人翻一条"变成"AI同时出三条"。

需要一张T10蓝牙耳机的白底产品图作为video skill的product_ref。提示词：


这个出一张就够——三个语种版本的视频都用同一张产品图做product_ref。

## 三语种视频详细设计

### 印尼语版 · TikTok Shop印尼风格

**市场背景**：印尼是东南亚最大电商市场，TikTok Shop渗透率极高。印尼消费者偏好：口语化、亲切、不过度精致、喜欢"真实试用"而非华丽广告。

**主播设定**：
- 性别：女性
- 年龄：25岁左右
- 外形：东南亚审美——健康小麦肤色、深棕色长直发、圆脸柔和五官。妆容清淡（淡眉+裸色唇）
- 穿搭：雾霾蓝短袖T恤，简约日常
- 模板：T2甜美邻家（圆脸/圆眼/颧骨低圆润/鼻头圆润小巧）
- 场景：明亮的现代公寓书桌，窗外是热带绿植，自然光
- 音色：年轻女性，语速偏快但清晰，带一点自然气声，亲切友好

**口播脚本（印尼语，75-90字/15秒）**：

```
{0-3s hook}
"Gue tiap hari commute 2 jam naik TransJakarta. Berisiknya tuh udah minta ampun. Akhirnya nemu ini — Saiyin T10."

（我每天通勤两小时坐TransJakarta。那个噪音简直了。最后找到了这个——Saiyin T10。）

{3-7s 产品证明}
"ANC-nya gila sih. Begitu dicolok ke telinga, dunia langsung senyap. Padahal harganya cuma Rp 399 ribu."

（降噪效果太疯狂了。一塞进耳朵，世界直接安静了。而且只要399千印尼盾。）

{7-12s 体感反馈}
"Baterainya 30 jam total. Gue charge seminggu sekali doang. Terus ini tuh tahan keringet, dipake lari juga aman."

（电池总共30小时。我一周只充一次电。而且这个防汗，戴着跑步也没问题。）

{12-15s 推荐收束}
"Buat yang budget mepet tapi pengen noise cancelling beneran — serius, coba dulu ini."
```

**画面分镜（轻量）**：

| 时间 | 画面 | 产品保留点 |
|------|------|-----------|
| 0-3s | 主播在窗边书桌前，拿着充电仓入画。背景窗外绿植 | 充电仓白色圆柱形清晰 |
| 3-7s | 主播戴上一只耳机，表情从皱眉→舒展。近景特写 | 耳机入耳后的短柄角度 |
| 7-12s | 主播在跑步机上戴着耳机，汗珠但耳机稳固。切回书桌，充电仓旁一杯咖啡 | 耳机IPX5防水、轻量4.2g |
| 12-15s | 主播对镜头微笑，充电仓托在掌心，logo正对镜头 | 品牌logo清晰、充电仓完整 |


### 泰语版 · Lazada/Shopee泰国风格

**市场背景**：泰国消费者偏好：审美精致、有礼貌但不生硬、画面干净美观。Lazada是主流平台，视频风格比印尼更精致但比中国更放松。

**主播设定**：
- 性别：女性
- 年龄：26岁左右
- 外形：泰国审美——自然肤色偏白、黑长直发带轻微波浪、鹅蛋脸、妆容精致但不过浓（自然眉+珊瑚色唇）
- 穿搭：砖红色棉麻短袖衬衫
- 模板：T1清冷高级（鹅蛋脸/丹凤眼/颧骨平内收/高鼻梁挺直）
- 场景：干净的米色卧室，窗外柔光，MUJI风格简约
- 音色：年轻女性，语速适中偏慢，声音温柔有礼貌

**口播脚本（泰语，75-85字/15秒）**：

```
{0-3s hook}
"ทุกวันนั่ง BTS ไปทำงาน เสียงดังจนฟังเพลงไม่รู้เรื่องเลยค่ะ จนได้ลองตัวนี้ — Saiyin T10"

（每天坐BTS去上班，吵到听歌都听不清。直到试了这款——Saiyin T10。）

{3-7s 产品证明}
"ใส่ปุ๊บ เสียงหายปั๊บเลยนะ ANC เขาทำมาดีมาก เสียงเงียบสนิท ไม่ใช่แค่เบาลงนะ คือเงียบจริง"

（一戴上，声音瞬间消失。ANC做得非常好，是完全安静，不是只是变小。是真的安静。）

{7-12s 体感反馈}
"แบต 30 ชั่วโมง ชาร์จทีเดียวอยู่ได้ทั้งอาทิตย์ ใส่แล้วเบามาก 4 กรัมนิดๆ เอง ใส่ทั้งวันไม่เมื่อยหู"

（电池30小时，充一次用一整周。戴着非常轻，才4克多一点，戴一整天耳朵不累。）

{12-15s 推荐收束}
"ถ้ากำลังหาหูฟัง noise cancelling ดีๆ ในราคาเบาๆ ตัวนี้คือคำตอบเลยค่ะ"
```

**画面分镜（轻量）**：

| 时间 | 画面 | 产品保留点 |
|------|------|-----------|
| 0-3s | 主播坐在窗边沙发，充电仓从画面右侧滑入。柔光从窗户打在脸上 | 充电仓哑光白磨砂质感 |
| 3-7s | 主播戴上耳机→闭眼享受音乐的表情→切耳机侧面特写，短柄在耳垂下方 | 耳机短柄+入耳角度 |
| 7-12s | 充电仓打开放在床头柜上，旁白"30小时"。切换到主播搭BTS，戴耳机看窗外 | 充电仓LED指示灯、USB-C口 |
| 12-15s | 主播托着充电仓对镜头微笑，画面干净 | logo清晰、Saiyin字样 |


### 英语版 · Amazon/TikTok Global风格

**市场背景**：美国消费者偏好——自然Tech Review感、直接、幽默但不夸张。Amazon是主战场，TikTok Shop正在增长。英语市场对"价值感"敏感——$29.99的ANC耳机谁都想试试。

**主播设定**：
- 性别：男性（三条视频错开性别）
- 年龄：28岁左右
- 外形：Caucasian，深棕短发微卷，方脸轮廓分明，下颌线清晰，胡茬修整干净。蓝灰色眼睛
- 穿搭：炭灰黑T恤
- 模板：T3成熟御姐（男性映射：硬朗成熟、方脸宽颧、高挺鼻、法令纹自然浅→商务大叔感）
- 场景：现代简约工作台，MacBook旁边，窗外城市景观
- 音色：年轻男性，美式口音，语速正常偏快，直接但有亲和力

**口播脚本（英语，70-80字/15秒）**：

```
{0-3s hook}
"Twenty-nine bucks. That's what I paid for these. And honestly? I was ready to be disappointed."

（29块。我就是花了这么多钱买的这副耳机。说实话——我已经准备好失望了。）

{3-7s 产品证明}
"But you put these in — and it's actually quiet. Like, proper ANC. Not 'slightly less noise'. Silent."

（但你戴上——它是真的安静。是正经的ANC。不是"噪音少了一点点"。是安静。）

{7-12s 体感反馈}
"Thirty-hour battery, USB-C charging, and they're so light I forget I'm wearing them. Gym, commute, coffee shop — they just work."

（30小时续航、USB-C充电，轻到我忘记自己戴着它们。健身房、通勤、咖啡店——就是好用。）

{12-15s 推荐收束}
"If you want noise cancelling without spending a hundred bucks — these are worth a shot. Saiyin T10."
```

**画面分镜（轻量）**：

| 时间 | 画面 | 产品保留点 |
|------|------|-----------|
| 0-3s | 主播在办公桌前，充电仓从桌面拿起。镜头从MacBook屏幕方向拍过来 | 充电仓圆柱形比例 |
| 3-7s | 主播戴上耳机→手指在耳边"snap"一下→耳机降噪指示灯亮起→主播表情"whoa" | 耳机短柄+金色充电触点 |
| 7-12s | 快速切换三个场景：跑步机（防水）+地铁（降噪）+咖啡店（轻量佩戴） | 耳机在运动中是否稳固、颜色保真 |
| 12-15s | 耳机+充电仓放在桌面上，旁边放价格标签"$29.99"，主播最后一句看向镜头 | logo+产品全貌 |

| 翻译口语化 | 印尼语/泰语/英语是否像当地人说话，而非机翻念稿 | 🔴 |
| 主播不撞脸 | 三条视频的主播是否确实是三个不同的人（不同性别/人种/模板） | 🟡 |
| 主播审美匹配市场 | 印尼主播像印尼人、泰国主播像泰国人、英语主播像欧美人 | 🟡 |
| 口播字数控制 | 各语种15秒字数是否符合UGC路线的speech_pace约束 | 🟡 |
| hook策略差异化 | 三条视频的hook是否不同（痛点/场景/反差） | 🟢 |
| 命名归档正确 | 文件名是否按"产品_语种_日期"规则 | 🟢 |
