## 一句话案例和痛点

**一句话**：老师把语文课文扔给豆包，豆包自动分镜→逐镜写生图提示词→生成静态图→转动态视频→加旁白配乐，输出一段可直接用于课堂播放的科普短片。

**普通人痛点**：做课件配图时网上搜的图不匹配课文细节（海参不是那样爬的、梭子鱼不是海豚），自己写提示词出来太卡通、太泛、没科学味。即使用了 AI 生图，也不知道怎么把静态图变成能吸引学生的动态视频——分镜要拆、镜头运动要写、旁白要配、剪辑要对节奏，一个老师根本搞不定整条链路。

## 造假背景与附件

**人物**：李老师，小学四年级科学老师，32岁。会用电脑但不会写 prompt。试过自己用 AI 生图，出来的效果"像幼儿园绘本，学生看了反而更困惑"。也试过把图喂给视频工具，但不知道怎么写镜头运动提示词，出来的视频要么全程推拉镜头毫无节奏，要么动物运动方向搞反了。

**附件**：课文《海底世界》全文（人教版小学语文四年级下册）
```
可知道，大海深处是怎样的吗?

海面上波涛澎湃的时候，海底依然很宁静。最大的风浪，也只能影响到海面以下几十米深。阳光很难射进深海，水越深光线越暗，五百米以下就全黑了。在这一片黑暗的深海里，却有许多光点像闪烁的星星，那是有发光器官的深水鱼在游动。

海底是否没有一点儿声音呢?不是的。海底的动物常常在窃窃私语。你用水中听音器一听，就能听见各种声音：有的像蜜蜂一样嗡嗡，有的像小鸟一样啾啾，有的像小狗一样汪汪，有的还好像在打鼾……它们吃东西的时候发出一种声音，行进的时候发出另一种声音，遇到危险还会发出警报。

海里的动物，各有各的活动方法。海参靠肌肉伸缩爬行，每小时只能前进四米。有一种鱼身体像梭子，每小时能游几十千米，攻击其他动物的时候，比普通的火车还快。乌贼和章鱼能突然向前方喷水，利用水的反推力迅速后退。还有些贝类自己不动，却能巴在轮船底下做免费的长途旅行。

海底的植物差异也很大。它们的色彩多种多样，有褐色的，有紫色的，还有红色的。最小的单细胞海藻，要用显微镜才能看清楚。最大的海藻长达二三百米，是地球上最长的生物。

海底蕴藏着丰富的煤、铁、石油和天然气，还有陆地上储量很少的稀有金属。

海底真是个景色奇异、物产丰富的世界。
```

## 模式A · 七轮共创对话脚本（静态图→动态视频→成片）

### 第一轮：给素材，说目标，不教方法

李老师：
```
我教小学四年级科学，有一篇课文《海底世界》，我想把它的内容做成一段视频，上课用。
之前我自己试过AI生图，但出来的东西太卡通、不科学——海参不知道画成了什么，梭子鱼画得像海豚，深海画得跟游泳池一样亮。
也试过把图转成视频，但镜头一直推推拉拉没有节奏，动物运动方向也是错的。
课文原文是这样的：
（粘贴课文全文）
你觉得这个该怎么弄？先不用急着出图，帮我理理思路。
```

**豆包应该做的**：理解内容类型（科普课文→教学视频）、受众（小学生）、当前失败原因（静态图事实错误+动态视频镜头无节奏+运动方向错误）。确认 16:9 横版、写实科普风格。识别出课文包含 8 个独立知识点，最终交付物是一段 60 秒左右的科普短片。

### 第二轮：补充约束，锁定底线

李老师：
```
对了，视频是要投影到教室白板上的，16:9 横版。
风格必须是写实科普——不是绘本，不是卡通。画面要能让学生看到"真实的海底是什么样"。
最重要的是：所有海洋生物的行为方式必须跟课文一致。海参就是靠肌肉伸缩爬行的，章鱼乌贼就是喷水后退的，不能画错，更不能在视频里动错方向。
```

**豆包应该做的**：确认比例和风格底线。开始提取课文中不可虚构的「绝对锁定」事实（海参肌肉伸缩爬行、梭子鱼时速、乌贼喷水后退、贝类巴船底、海藻色彩等）。

### 第三轮：让豆包先诊断后分镜

李老师：
```
你觉得这课文应该拆成几个镜头？每个镜对应哪一段知识点？
不用写完整的提示词，先把分镜框架列出来——我看看逻辑对不对。
对了，我之前出图失败和视频失败的原因到底是什么？你也帮我分析一下。
```

**豆包应该做的**：
1. 诊断失败原因：
   - 静态图失败：没有分镜（一条 prompt 塞太多）、没有锁定事实（模型随便画）、风格描述太空（"好看""高级"→无所适从）
   - 视频失败：镜头运动没有跟内容节奏走（全程推拉=没有节奏）、运动方向没有锁定（AI不知道海参往哪爬）、没有给每镜设定"动什么、不动什么"
2. 给出分镜框架：8 镜，每镜对应课文一段，镜1=海面与深海对比，镜2=发光鱼群，镜3=海底声音，镜4=海参vs梭子鱼，镜5=乌贼喷水+贝类巴船，镜6=植物尺度对比，镜7=海底矿产，镜8=大全景收尾
3. 标注每镜的运动类型（静镜/动镜/半动镜）

### 第四轮：逐镜写静态图提示词

李老师：
```
分镜可以。现在给每一镜写完整的生图提示词——要详细。
每一镜都要包含：画幅比例、课文原文对应的事实（绝对不能画错的部分）、这镜让人记住什么、具体构图、光影和材质、文字怎么处理、最容易跑偏的点明确禁止。
写得越详细越好。现在只写提示词，不执行。
```

**豆包应该写的**（以镜4为例，应覆盖全部8镜）：
```
【任务与比例】16:9 横版科教插图
【绝对锁定】课文原文"海参靠肌肉伸缩爬行，每小时只能前进四米。有一种鱼身体像梭子，每小时能游几十千米，攻击其他动物的时候，比普通的火车还快。"物种类别和行为方式不可偏离：海参必须有肌肉伸缩的体态；梭子鱼必须是梭形流线体，不能画成海豚、鲨鱼或金枪鱼。
【创意母题】"一小时四米 vs 比火车还快"
【主体、场景与构图】左侧前景：海参趴在海底沙地上，身体有明显的肌肉伸缩褶皱，身下沙面有刚刚爬过的浅痕。右侧：一条流线型梭子鱼从右侧入画，身后拖出细密气泡轨迹，身体呈蓄力冲刺姿态。背景用轻微动感模糊暗示速度差异。
【材质、光影与色彩】海底沙地暖灰色。海参深褐带肉感纹理。梭子鱼银蓝金属光泽。两侧光照一致，从上方斜射。两主体在画面中的间距预留梭子鱼冲刺空间。
【文字/后期策略】不出现任何文字，纯画面。
【系列衔接】左侧海参位置保持不变，方便后续镜4的动态视频中作固定锚点。右侧预留大量负空间给梭子鱼的运动轨迹。
【针对本任务的禁止项】禁止海参画成海螺、海胆或海星；禁止梭子鱼画成海豚或鲨鱼
```

### 第五轮：静态图执行 + 输出确认

李老师：
```
提示词可以。现在按这个执行，用 Seedream 生图，每镜独立生成一张 16:9 横版。
先出图，别做视频。出完我要对照课文检查一遍。
```

**豆包应该做的**：逐镜执行生图，输出 8 张静态图。不在此轮产生视频。

### 第六轮：静图转动态视频 + 逐镜镜头指令

李老师：
```
图都看过了，没问题。现在把这些静态图转成视频——要给我的学生看的，不是给评委看的。
每镜写视频生成提示词。注意：

1. 不要每镜都推拉摇移——有些镜需要"静"（比如看发光鱼群、看海底矿产），有些需要"动"（海参爬行、梭子鱼冲刺、乌贼喷水后退）。画面节奏要跟课文的叙述节奏走。
2. 镜3（海底声音）的处理要想清楚：课文说的是"窃窃私语"，画面上看不到声音，但视频里可以通过极缓慢的水流波纹+微弱的发光生物闪烁来暗示声音的存在。不要画文字、不要画音符。
3. 所有运动必须符合课文描述的方向——海参是从左往右缓慢爬行，梭子鱼是从右往左高速掠过，乌贼是向前喷水身体后退。
4. 每镜时长控制在 5-7 秒，总片长控制在 60 秒以内。太快学生跟不上，太慢走神。
5. 镜与镜之间的转场也要写清楚——硬切还是叠化、从哪个元素过渡到哪个元素。

先写提示词，不执行。
```

**豆包应该写的**（以镜4为例，其余7镜同样结构）：
```
【镜4 · 慢与快 | 时长 6 秒】
【绝对锁定】海参肌肉伸缩的体态不可中断或变形；梭子鱼保持梭形。运动方向：海参从左向右缓慢移动，梭子鱼从右向左高速掠过。
【镜头运动类型】全程固定机位——不推、不拉、不摇。靠被摄物自身的速度差制造节奏。
【动态时间线】
  0.0-1.0s：开幕即见海参在左侧沙地上，身体正在做一次肌肉伸缩，前半段收缩、后半段伸展
  1.0-1.5s：海参完成一次爬行循环，沙面留下新的浅痕
  1.5-2.0s：画面右侧远处出现一个微小银色光点（梭子鱼入画前兆）
  2.0-2.3s：梭子鱼从右侧高速入画，身后拖出气泡轨迹，画面轻微动感模糊
  2.3-2.6s：梭子鱼穿过画面中央，沙地上方产生扰动波纹
  2.6-3.0s：梭子鱼从左出画，气泡轨迹逐渐消散
  3.0-6.0s：画面恢复宁静，海参继续缓慢爬行，沙地上的新痕与旧痕形成对比。最后 1 秒画面微暗作为转场前奏
【转场】叠化至镜5（乌贼喷水），叠化点选在海底沙地的暖灰色纹理延续上
【音效建议】（仅供后期参考，此阶段不出声）海参爬行极微弱的沙粒摩擦声；梭子鱼掠过时短暂的水流撕裂声
【针对本任务的禁止项】禁止摄像机有任何移动；禁止梭子鱼变形为海豚；禁止海参的肌肉伸缩做成蠕虫式蠕动
```

### 第七轮：配音 + 成片输出

李老师：
```
镜头的提示词都可以。现在做最后一步——给整段视频配上旁白和背景音。

旁白就用课文原文，每镜对应的那段文字念出来就行。不用加额外解说、不用改写成"生动活泼"的语气——课文原文本身就是最好的旁白。

背景音要有海底的氛围感：低频的水流声、偶尔的气泡声、微弱的海底生物鸣叫。不要喧宾夺主——旁白是第一信息源，背景音只是氛围。

输出说明：
1. 先确认每镜旁白和画面的时间对齐——旁白念到"海参靠肌肉伸缩爬行"时，画面上正好是海参在伸缩
2. 总时长控制在 60 秒左右
3. 片头加 2 秒黑屏+课文标题"海底世界"，片尾加 2 秒黑屏收束

执行，直接出成片。
```

## 独立检测真值（豆包不会被提前喂的内容）

验证者用以下标准独立判断产出是否合格，无需对照豆包的提示词：

### 静态图验收

| # | 检查项 | 通过条件 | 权重 |
|---|--------|---------|------|
| 1 | 海参爬行方式 | 可见肌肉伸缩的体态细节，不是滑行/蠕动 | 致命 |
| 2 | 梭子鱼形态 | 梭形流线体，不是海豚/鲨鱼/金枪鱼 | 致命 |
| 3 | 乌贼运动方向 | 有明显喷水轨迹+后退姿态，不是向前游 | 致命 |
| 4 | 深海光照 | 500米以下无环境光，只有发光生物的光点 | 致命 |
| 5 | 海藻色彩 | 出现褐色/紫色/红色中至少两种，不是全绿 | 重要 |
| 6 | 巨藻尺度 | 有多层叶片的明确主干结构，不是普通海草 | 重要 |
| 7 | 贝类状态 | 巴在船底，有附着姿态，不是浮在水中 | 重要 |
| 8 | 风格一致性 | 8图的光方向、细节密度、色调统一 | 重要 |
| 9 | 无乱码文字 | 没有一张图出现 AI 生成的乱码文字 | 次要 |
| 10 | 无水印/卡通 | 没有任何卡通化处理或风格化滤镜 | 次要 |

### 动态视频验收

| # | 检查项 | 通过条件 | 权重 |
|---|--------|---------|------|
| 11 | 运动方向正确 | 海参从左→右，梭子鱼从右→左，乌贼向前喷水后退 | 致命 |
| 12 | 镜头节奏 | 不是全程推拉——静镜确实静（如发光鱼群），动镜确实动（如梭子鱼冲刺） | 致命 |
| 13 | 镜3声音暗示 | 无水波纹动画、无卡通音符、无文字——只用极微弱的生物光闪烁+水流暗示"窃窃私语" | 重要 |
| 14 | 旁白时序对齐 | 旁白念到"海参爬行"时画面确为海参镜，"梭子鱼"时确为梭子鱼镜 | 重要 |
| 15 | 旁白风格 | 直接念课文原文，不改编、不转述、不加"生动解说" | 重要 |
| 16 | 转场匹配 | 硬切或叠化的接点有视觉逻辑（颜色/方向/元素延续），不是随机切 | 次要 |
| 17 | 片头片尾 | 2秒黑屏+标题入场，2秒黑屏收束 | 次要 |
| 18 | 总时长 | 55-65 秒之间 | 次要 |
| 19 | 背景音 | 旁白音量始终高于背景音，无可辨识的旋律（避免版权风险） | 次要 |

## 视觉创作专项（区别于数据看板/网页应用类案例）

本案例属于视觉创作场景，且最终交付为视频，需额外关注：
- **事实正确 > 画面美感**：画面再漂亮，海参画错了、动错了方向就是废片
- **逐镜分镜**：一条 prompt 塞 8 个知识点=每个都糊，必须拆镜
- **禁止项必须具体**：不说"不要丑"，说"禁止把梭子鱼画成海豚""禁止摄像机做推拉运动"
- **文字默认后期**：AI 写字不可靠，除片头标题外一律后期
- **视频是本案例的最终交付物**：静态图只是中间产物，最终价值在成片。达人拍摄时需同时展示"豆包写出逐镜镜头提示词"和"从静态图到动态视频的转变"
- **旁白=课文原文**：不改编、不润色——课文本身就是最佳解说词。降低了后期录制旁白的门槛，也避免了"AI 改写后偏离原文"的风险
- **镜头运动服务于内容，不炫技**：视频范例中最有说服力的不是推拉镜头，而是"全程固定机位，靠海参和梭子鱼自身速度差制造节奏"这种设计思路
- **风格一致性**：同一套图的色板、光向、细节密度保持不变；视频转场有视觉逻辑
