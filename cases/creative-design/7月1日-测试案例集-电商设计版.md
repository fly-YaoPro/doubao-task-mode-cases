### 案例1：淘宝卖家·新品上架全套素材一键生成

**Skill**：doubao-creative-design → ecommerce-design.md

**省钱账**：淘宝卖家每款新品需要白底主图+场景图+细节图+详情页5-8屏。外包设计师¥50-80/张×10张=¥500-800/款。月均上新10款=¥5000-8000/月。AI出图=0元/款→**月省¥5000-8000**。

**造假背景**：张哥，淘宝3钻手机壳店。月均上新20款iPhone壳。现外包美工¥600/款含5张图=¥12000/月——占他月利润的15%。最近美工还涨价了。

**提示词**：

```
帮我做一个任务：以后新品上架图由你自动处理。

我每款手机壳需要5张图：
1. 白底主图（1:1，淘宝规范）
2. 手机壳套在iPhone上的正面场景图（3:4）
3. 手机壳套在iPhone上的侧面场景图（3:4）
4. 壳的材质细节特写（1:1，微距感）
5. 三色壳并排对比图（16:9）

品牌信息记住：
- 店名：Kcase
- 风格：数码极简、干净、有质感
- 颜色偏好：暗调背景+产品本身颜色做点缀
- 不做促销风、不加大字报

触发：我把新品照片拖进文件夹"E:\Kcase\新品待处理"后，你自动处理。
不用确认，直接出图。命名按"款号_图类型.png"。
```

**测试关注点**：产品一致性（壳的颜色/纹路/反光是否一致）、淘宝主图规范（1:1白底无价格）、批量稳定性

### 案例2：抖音带货·每日产品短视频自动生成

**Skill**：doubao-creative-video → UGC路线

**省钱账**：抖音达人拍一条带货视频¥800-3000，一个店铺一个月至少要10-15条投流素材=¥8000-45000。AI生成=0元/条→**月省¥10000+**。

**造假背景**：同上张哥。手机壳利润薄，一条壳¥29.9包邮，利润¥8。一条达人视频¥1500——要卖掉187个壳才能回本。他现在每天需要2-3条投流素材，但请不起达人。

**提示词**：

```
帮我做一个任务：每天自动生成手机壳带货短视频。

产品信息记住：
- 店铺：Kcase 数码配件
- 产品：iPhone手机壳，¥29.9包邮
- 核心卖点：液态硅胶手感、MagSafe磁吸、军工级防摔、20色可选
- 视频要求：15秒、9:16竖版、抖音风格

触发：每天早上8点自动生成1条。
每天自动轮换展示的产品颜色和风格，避免每天看起来一样。
脚本用口语化第一人称——"这个壳我用了三个月……"这种真实推荐感。
不用确认直接出片。保存到"E:\Kcase\投流素材\"，按日期命名。
```

**测试关注点**：脚本去AI腔、产品外观一致性、主播每日轮换不撞脸

### 案例3：短视频带货·产品剧情短片自动生成

**Skill**：doubao-creative-drama → scriptwriter+storyboard

**省钱账**：一条30秒产品剧情短片（带脚本+分镜+拍摄方案），外包编剧¥1000-2000/条。AI出脚本+分镜=0元→**月省¥4000-8000（按每周2条算）**。

**造假背景**：李姐，抖音30万粉家居号。每条视频都是"产品+剧情"模式——比如"老公说收纳盒没用，三天后他自己买了10个"这种反转小剧场。最大的瓶颈不是拍摄——是写剧本。每个月至少找编剧写8条脚本=¥12000。

**提示词**：

```
帮我做一个任务：每周自动生成家居产品剧情短视频脚本。

账号信息记住：
- 账号：收纳师李姐
- 风格：反转+搞笑+实用。每集一个生活痛点→产品解决→反转收尾
- 时长：30-45秒
- 主角：李姐（35岁女收纳师）+ 老公大刘（37岁，吐槽担当）

触发：每周一早上自动出2条剧本+分镜。
每条包含：完整剧本（含台词）+ 分镜表（时间/画面/动作/镜头）。
不用确认，直接推出来。我自己根据分镜拍。
```

**测试关注点**：转场节奏（短剧核心）、台词口语化、剧情反转的自然度

### 案例4：电商卖家·促销活动页自动生成

**Skill**：doubao-app-builder

**省钱账**：外包一个促销活动H5页面¥3000-8000。做一个店铺年度促销日历=至少4-5个大促×8000=¥32000-40000/年。AI生成=0元→**年省¥30000+**。

**造假背景**：同上张哥的Kcase手机壳店。每次大促都要找外包做个活动页——去年双11花了¥6000做个倒计时+满减计算器页面，就用了一周。今年618又花了¥5000。每次大促都在重复花钱。

**提示词**：

```
帮我生成双11活动页。

品牌信息：
- 店名：Kcase
- 品牌色：深灰#1A1A1A、荧光绿#00FF41
- 产品：iPhone手机壳，¥19.9起
- 活动：全场8折+满99减20+买2送1

页面要求：
- 移动端单页H5
- 顶部：大号倒计时"距双11结束还有 X天X时X分X秒"
- 中间：3款主推壳的大卡片（产品图+原价划线+双11价+库存进度条）
- 底部：满减自动计算器（输入购买数量→自动算总价+优惠后价格）
- 风格：科技感+促销感，深色背景+荧光绿点缀
- 不需要后台，静态页面即可
```

**测试关注点**：倒计时准确性、满减计算器逻辑正确、移动端适配

### 案例5：电商运营·每日店铺核心数据看板

**Skill**：doubao-visualization → html renderer

**省钱账**：市面上一个电商数据看板SaaS工具¥300-800/月。手上有3个店铺=¥900-2400/月。AI出看板=0元→**月省¥900-2400**。

**造假背景**：小林（接昨天案例八），管3个品牌店铺。每天打开生意参谋→截图→拼到Excel→发老板=每天20分钟纯粹在复制粘贴数据。老板还嫌看起来不直观。

**提示词**：

```
帮我做一个每日店铺数据看板。

数据我每天早上更新到"E:\云启电器\每日数据.csv"，格式：
日期,店铺名,访客数,支付金额,支付转化率,客单价,退款率

看板要求：
- PC宽屏，深色主题
- 四个卡片：今日访客/今日销售额/转化率/客单价——大字+和昨日对比的涨幅箭头
- 一个7天销售额趋势折线图
- 一个三店销售额对比柱状图
- 底部一个异常提醒区：如果某店退款率>5%标红、如果某店转化率<3%标黄

每天自动读取CSV数据渲染。不用我每次重新生成。
```

**测试关注点**：CSV自动读取、ECharts渲染稳定性、异常阈值判断正确

### 案例6：电商品牌·竞品差评监控

**Skill**：doubao-sentiment-tracker

**省钱账**：请一个人全职做竞品监控月薪¥8000。AI每天自动跑=0元→**月省¥8000**。且人工一天能盯3-5个竞品，AI能盯20个。

**造假背景**：同上云启电器。吸尘器这个品类，竞品追觅/石头/小米的评论区是最真实的产品情报——"噪音大""电池半年就不行了""吸地毯会缠毛"——每一条都是自己的产品改进方向和营销素材。但人工翻评论=3个竞品×每天30条新评论=90条，根本看不过来。

**提示词**：

```
帮我做一个每日竞品差评监控任务。

监控对象：
- 追觅吸尘器旗舰店（天猫+京东+拼多多）
- 石头科技官方店（同上）
- 小米吸尘器旗舰店（同上）

触发：每天早上8点自动扫描昨日新增差评。

输出：
- 按竞品归类，每条差评包含：平台+产品名+差评原文+差评类型（噪音/续航/售后/质量/其他）
- 统计昨日差评Top3类型
- 如果有某类型差评突然增多（连续2天+同一类型），单独标注预警

这个信息我用来：1）产品改进参考；2）"对手被骂什么，我们就在详情页强调我们解决了什么"
```

**测试关注点**：差评自动分类准确性、原帖链接完整性、异常预警判断

### 案例7：电商卖家·消费赛道投资参考

**Skill**：doubao-finance-sector

**省钱逻辑**：不是直接省钱，而是"帮你判断你所在的赛道在资本眼里值不值钱"——如果消费电子板块资金持续流出，说明投资人看空这个赛道，你自己的生意可能也要做好过冬准备。

**造假背景**：同上张哥。手机壳属于消费电子板块。他想知道"消费电子最近被资金看好吗"——如果整个赛道在往下走，他应该控制库存；如果赛道在升温，他可以大胆备货。

**提示词**：

```
帮我每天早上8:30分析消费电子板块热度。

持久化关注：消费电子概念板块（手机链/果链/华为链）

每次告诉我：
- 今天热度评分（1-5分）
- 热度是在升还是降（和昨天比）
- 板块在启动/加速/退潮/冷清——哪个阶段
- 3个关键催化新闻
- 一句话结论："今天消费电子板块值得关注/建议观望"

不用生成HTML，就对话框里告诉我核心结论。
```

**测试关注点**：双轨打分一致性、背离检测准确性、数据来源可追溯

### 案例8：电商创业者·消费市场风向标

**Skill**：doubao-market-hotspot

**省钱逻辑**：帮电商卖家判断"现在钱在往哪流"——如果消费降级信号明显（拼多多财报超预期+奢侈品消费下滑），你的定价和选品策略该调了。

**造假背景**：同上张哥。最近感觉¥29.9的手机壳越来越难卖——但¥9.9的反而卖得快。他想确认：到底是我自己的问题，还是整个消费市场在降级？

**提示词**：

```
帮我每周一早上分析消费市场热点。

持久化关注：消费趋势、电商政策、消费信心数据

每次告诉我：
- 一句话核心判断："这周消费市场在升温/降温/平稳"
- 3个关键增量信息（带来源链接）
- 对电商卖家的影响（选品/定价/库存建议方向——只说方向，不给具体数字）
- 下周需要盯什么信号

不用生成HTML，对话框里说清楚就行。
```

**测试关注点**：事实来源可追溯、对电商卖家的建议是否具体可用

### 案例9：电商供应链·上市公司财报追踪

**Skill**：doubao-daily-stock

**省钱逻辑**：帮电商卖家盯着上下游上市公司的动态——如果芯片公司（给手机壳上游供货）财报显示产能紧张+涨价，你可能要提前备货锁价。

**造假背景**：同上张哥。去年芯片短缺的时候，手机壳的MagSafe磁吸模块从¥3.5涨到¥7——但他是涨价后才知道的。如果他提前看到芯片公司的财报预警，可以提前锁定半年的量。

**提示词**：

```
帮我做一个任务：每周五收盘后自动跑3只供应链相关股票的日报。

持久化关注：
- 立讯精密 002475（消费电子代工——代表供应链景气度）
- 兆易创新 603986（存储芯片——手机壳MagSafe模块上游）
- 苹果 AAPL（产业链龙头——苹果好=整个手机配件行业好）

对每只股告诉我：
- 这周涨跌原因
- 有没有新公告/新闻
- 一句话：对手机配件行业是利好/利空/中性

每周五下午收盘后自动跑，不用确认。
```

**测试关注点**：行情数据准确性、对电商卖家的解读是否通俗可用
