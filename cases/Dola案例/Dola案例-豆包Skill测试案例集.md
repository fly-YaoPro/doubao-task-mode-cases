## 📌 案例一：旧物/废片一键变广告级产品图

### 造假背景

**人物**：王姐，35岁，全职妈妈，闲鱼副业卖家。手里攒了十几个二手轻奢包、十几双孩子穿小的品牌童鞋。

**困境**：在卧室床上拍的包——黄光+碎花床单背景+旁边奶瓶。挂了两周没人问。想花200块请摄影师拍，但一个二手包的利润就300块，划不来。也试过某电商平台的一键白底工具——出来的图像抠图贴白纸，边缘毛糙，包的皮革纹理全丢了。

**真实痛点**：二手非标品拍照成本占利润的50%以上，但不拍好图就卖不掉。手机拍的不专业，找摄影师不划算。

### 1.1 场景测试反馈

**测试场景**：闲鱼二手奢侈品 → 白底主图+场景图+上身图

**突出用法（区别于普通模式）**：
doubao-creative-design的I2I能力不是简单的"AI去背景"，而是：
1. **产品一致性锁**：基于上传原图做edit，保留包的monogram花纹、皮革纹理、五金件光泽——不是重画一个新包，而是把你的包放进专业影棚
2. **多图串联一致性**：同一次任务生成的所有图（白底/场景/上身）都基于同一张上传原图，确保跨图产品外观不漂移
3. **影子/光影融合**：产品在新场景中的阴影方向、环境反光与环境光一致，不会出现"PS贴上去"的生硬感

**提示词（优化版）**：

```
上传这只LV Neverfull中号包的实拍照片。

帮我做三张图，全部基于这张照片做图生图(I2I)，保持包的颜色、材质、花纹不变：

【图一：淘宝风格白底主图】
- 保留原图的包型轮廓、monogram花纹颜色、皮革纹理和五金件细节
- 将杂乱的卧室背景替换为干净纯白底（#FFFFFF）
- 包下方添加自然投影，光线从左上角45°打来，柔光质感
- 比例1:1，淘宝主图标准

【图二：场景展示图·高端梳妆台】
- 保留包的完整外观（颜色、材质、花纹、五金不变）
- 替换背景为：白色大理石台面梳妆台，左侧自然晨光，窗帘半透明
- 包放在台面正中央，旁边放一支口红（Tom Ford金色管）和一副墨镜，简约不抢戏
- 光线：上午10点窗边自然光，包边缘有柔和高光
- 比例3:4

【图三：上身场景图·咖啡店】
- 保留包的外观不变
- 一位30岁亚洲女性，穿米色风衣，手持这个包，坐在咖啡店靠窗位置
- 窗外自然光，背景轻微虚化，人物面部清晰自然
- 包的尺寸比例真实——中号Neverfull，人手持比例正确
- 比例9:16
```

**测试关注点**：

| 关注项 | 描述 | 风险等级 |
|--------|------|---------|
| 产品一致性-花纹 | monogram是否与原图一致（最高风险） | 🔴 高 |
| 产品一致性-颜色 | 皮革颜色在不同场景光照下是否保持原色 | 🟡 中 |
| 边缘融合 | 包在新场景中的边缘是否有"抠图白光" | 🔴 高 |
| 光影方向 | 三张图的光源方向是否与场景统一 | 🟡 中 |
| I2I链路 | 三张图是否均基于原图做edit（非独立文生图） | 🔴 高 |
| 文字渲染（如盒上logo） | 包身上的品牌文字是否保持原样 | 🟢 低 |

### 1.2 场景发散

- **二手数码**：碎屏手机→修复+换场景；iPhone实拍→亮屏+多角度（闲鱼转转年交易额超千亿）
- **家具转让**：仓库随手拍折叠桌→客厅/书房展示图（同城二手群）
- **农产品电商**：地里拔的带泥胡萝卜→白底+厨房场景（拼多多助农）
- **球鞋转卖**：穿过的AJ→去污+补磨损+潮流场景（得物/闲鱼）
- **文玩珠宝**：随手拍的翡翠→黑丝绒+聚光+微距（微拍堂）
- **汽车二手车**：路边拍的私家车→展厅大片效果（二手车平台）


## 📌 案例二：一张产品图→小红书/朋友圈/抖音三平台海报矩阵

### 造假背景

**品牌**："唤醒WakeUp"，新创冷萃咖啡液品牌（虚构）。创始人小陈，前互联网PM，辞职创业做咖啡。只有一个SKU——冷萃咖啡液10条装（每条约15ml）。

**困境**：自己用iPhone拍了包装盒照片，需要小红书种草图+朋友圈预热海报+抖音信息流投放素材。问了一家设计工作室报价：小红书海报800×2 + 朋友圈500 + 抖音投放素材700 = 2800，7个工作日。但新品上线就在三天后。

**真实痛点**：新消费品牌前期完全没有设计预算（钱都花在产品上），但社媒内容又必须好看——这是鸡生蛋蛋生鸡的死循环。

### 2.1 场景测试反馈

**测试场景**：新消费品牌 → 三平台海报矩阵

**突出用法（区别于普通模式）**：
1. **品牌一致性锁**：第一次生成的包装盒版本成为核心资产，后续所有平台海报基于此做edit——确保三张图的产品外观、颜色100%一致
2. **平台视觉指纹内置**：Skill内置了小红书（杂志感柔焦+手账排版）、朋友圈（第一人称手持+轻写标注）、抖音（破屏大字+强对比）的视觉规范——不需要用户自己去描述"小红书是什么风格"
3. **一张图→多尺寸多风格输出**：不是"每次重新描述一张海报"，而是"基于这个产品，按三个平台的规范各出一张"

**提示词A · 小红书种草风**：

```
上传产品包装盒照片。

帮我生成一张小红书风格的产品种草海报。目标：让刷到的女生觉得"这个必须试试"。

产品信息：
- 品牌：唤醒WakeUp
- 产品：冷萃咖啡液10条装（一条15ml，一条兑一杯）
- 包装：白色纸盒、橙色#FF6B35品牌色腰封、盒面有"冷萃咖啡液"字样
- 核心卖点：10倍浓缩、0糖0脂、3秒即溶（冷热水都行）
- 售价：69元/盒
- 面向人群：25-35岁女性，早起通勤/办公室困了来一杯

画面要求：
- 比例3:4竖版
- 场景：日系简约厨房，清晨7点的自然光从窗户斜入
- 构图：厨房岛台上，产品包装盒放在画面右下1/3处，旁边一杯刚冲好的冰美式（透明玻璃杯，能看到冰块、咖啡液下沉的层次、冷凝水珠）
- 远景：岛台上有几颗咖啡豆、一支铜色手冲壶，但不抢主体
- 风格：温暖生活感、有呼吸感。画面留白约30%，不要塞满元素
- 字效：左下角一小段手写体文字"早八续命水，15秒搞定"，颜色#333
- 保持包装盒的颜色、logo和细节与上传照片一致
- 不要出现价格、二维码
```

**提示词B · 朋友圈预热海报**：

```
基于同一张上传的包装盒照片，做一张朋友圈预热海报。

要求：
- 比例1:1正方形
- 场景：干净米色桌面+俯拍（Flat Lay），极简构图
- 产品放在中央偏上位置，周围散落几颗咖啡豆和一片橙子干（不要太整齐）
- 主文案："明天见。" ——大字，细黑体，居中，字号占画面高度的1/6
- 副文案："冷萃咖啡液·唤醒WakeUp" ——小字，在"明天见"下方
- 品牌logo在底部居中
- 风格：克制、高级、有留白。像一张有设计感的电影海报，不像促销广告
- 色调：暖米色为主，橙色作为点缀色
- 不要出现价格、折扣
```

**提示词C · 抖音信息流投放图**：

```
基于同一张上传的包装盒照片，做抖音信息流投放素材。

要求：
- 比例9:16竖版
- 上半部分（占画面2/3）：咖啡液倒入冰水中的高速摄影感画面——深褐色咖啡液在透明玻璃杯的水和冰之间层层扩散，视觉冲击力强
- 下半部分（占画面1/3）：深色底+白色大字
  主标："太困了，但不想喝糖水" ——白色粗体，字号大
  副标："冷萃咖啡液·真正0糖·3秒搞定" ——白色细体，稍小
- 角落："新品首发·第二盒半价" ——小字标签，橙色底白字
- 保持包装盒外观（在上传图中出现的版本）与A/B两图一致
- 风格：干净、对比强、在信息流刷到时3秒能抓住注意力
```

**测试关注点**：

| 关注项 | 描述 |
|--------|------|
| 跨图一致性 | 三张图的包装盒颜色、尺寸、文字是否一致（同一核心资产） |
| 平台视觉准确性 | 小红书是否够"种草感"，朋友圈是否够"日常感"，抖音是否够"冲击感" |
| 中文文字渲染 | 所有中文字是否清晰可读，无乱码、无变形 |
| 核心资产锁定 | 后两张是否基于第一张edit，而非独立文生图 |
| 产品自然度 | 包装盒在场景中的光影和透视是否自然融合 |

### 2.2 场景发散

- **餐饮外卖**：一道菜→美团/饿了么店铺装修图+小红书探店图
- **健身场馆**：场馆照→大众点评头图+小红书获客图+朋友圈活动海报
- **美容院**：项目实拍→抖音团购推广图+朋友圈案例图
- **课程/训练营**：一张封面→公众号封面+视频号封面+朋友圈海报三件套
- **线下市集/快闪**：摊位照→小红书预告+朋友圈邀请+现场返图海报


## 📌 案例三：一张产品图→15秒抖音带货短视频

### 造假背景

**公司**："赛音科技"（虚构），深圳宝安一家蓝牙耳机工厂。做OEM/ODM七八年，给外贸客户供货。2026年初注册自有品牌"赛音Saiyin"，第一款产品T10真无线降噪耳机上市。

**困境**：老板老周花3500块请了一个本地抖音达人拍带货视频。达人发过来一看——念稿僵硬、全程一个机位拍到结束、产品特写都跑焦了，完播率9%。老周觉得不如自己来。但他只会拍产线上的产品照片，不会拍视频。

**真实痛点**：白牌厂商的最大痛——产品质量没问题，但没有任何内容能力。淘宝详情图是1688模板，抖音视频是0。做品牌的第一步就卡在内容。

### 3.1 场景测试反馈

**测试场景**：白牌蓝牙耳机 → 15秒UGC带货短视频

**突出用法（区别于普通模式）**：
doubao-creative-video的UGC策略不是"把产品图变成视频"，而是：
1. **主播随机化**：每次生成自动抽不同的主播模板（5个模板×随机肤色/发色/年龄/穿搭），确保不同视频之间主播不撞脸——模拟"不同达人在种草你同一个产品"
2. **产品一致性锁**：所有镜头中的耳机外观（圆柱形充电仓、哑光磨砂白色）基于上传产品图锁定，不重绘
3. **安利脚本引擎**：根据产品卖点自动生成真人化口播台词——用"我自己用下来""一开始我也没抱期待"这类第一人称表达，避开AI的"宝子们冲"和机械的"此产品具有XX优势"
4. **15秒高分四段式**：0-3s开头钩子→3-7s产品证明→7-12s使用反馈→12-15s推荐+CTA

**提示词**：

```
帮我生成一个15秒的UGC口播短视频，推荐这款蓝牙耳机。用于抖音和快手投放。

产品信息：
- 品牌：赛音 Saiyin
- 型号：T10
- 品类：真无线降噪蓝牙耳机
- 外观：白色圆柱形充电仓（略粗于食指），哑光磨砂质感。耳机本体是入耳式，短柄设计
- 核心卖点：ANC主动降噪（千元级）、30小时总续航（含充电仓）、蓝牙5.4、IPX5防水防汗、佩戴舒适
- 价格：199元
- 定位语：千元级降噪，不到200块
- 目标用户：18-28岁学生和通勤族，坐地铁/自习室/健身使用
- 用场景：挤地铁、图书馆复习、健身房跑步

视频要求：
- 9:16竖版，15秒
- UGC真实博主推荐感——像朋友推荐，不像广告
- 有主播出镜口播（随机真实素人，不撞脸）
- 音色：年轻女性或男性，亲切自然，语速正常偏快但不急促
- 台词语感：第一人称体验式——"我一开始也觉得199块能有什么降噪""结果戴上那一刻整个世界都安静了""通勤两小时现在终于能好好听歌了"
- 必须出现产品至少3次：开场货架展示→产品特写→结尾hold
- 背景音乐：轻Chill Beats/Lo-fi，音量低于人声
- 场景：窗边书桌或咖啡店角落，自然光
- 风格：真实、不做作、有确信感。让人看完想搜一下多少钱
```

**测试关注点**：

| 关注项 | 描述 |
|--------|------|
| 产品一致性 | 充电仓是否始终保持圆柱形白色磨砂外观 |
| 脚本去AI腔 | 口播是否像真人说话（半句、转折、停顿感），不AI机械 |
| 画面信息量 | 15秒内是否包含多个镜头（开箱→佩戴→场景→hold） |
| UGC质感 | 画面是否像手机拍的而非棚拍广告 |
| 主播质量 | 人物是否自然上镜、是否自然地眨眼看镜头 |

### 3.2 场景发散

- **白牌美妆**：气垫/口红→15秒试用+效果对比
- **食品/零食**：包装→拆袋→试吃→推荐
- **小家电**：空气炸锅→功能演示+成品展示
- **宠物用品**：猫砂→使用前后对比+养猫人推荐
- **母婴**：宝宝辅食机→操作演示+安全性强调


## 📌 案例四：静态海报→5秒动态开屏广告

### 造假背景

**品牌**："树觉生活"（虚构），原创设计家居品牌，主打实木家具。主推新品"山脊"餐桌——黑胡桃木+极简锥形腿。

**困境**：设计师画好了618促销海报（餐桌场景+7折文案），但不能直接用在海报在抖音——静态图在短视频平台的完播率极低。找人做动画视频报价8000起步、排期10天。但618预热页面明天就要上线。

**真实痛点**：电商大促节奏下，海报做好了但"动不起来"。从海报到视频的转化成本太高，导致大部分商家只能拿静态海报硬投——效果差但也没别的办法。

### 4.1 场景测试反馈

**测试场景**：电商促销海报→5秒动态广告视频

**突出用法（区别于普通模式）**：
Dola Row 2的prompt定义了极其精细的5秒动画时间线——这不是一般AI工具能做到的"海报动起来"，而是：
1. **分层动画**：背景层先"醒"（0-1s）→产品入画（1-2.4s）→标题入场（2.4-3.3s）→logo/辅文入场（3.3-4s）→最终帧100%还原原海报（4-5s）
2. **零幻觉**：不新增粒子、不换场景、不加元素——最终帧是原海报的精确副本
3. **可用作开屏广告**：5秒的时长+9:16比例，直接适配抖音开屏/TopView格式

**提示词**：

```
上传这张家居品牌618促销海报。

帮我将这张静态海报转成5秒动态广告视频。

要求：
- 最终帧（5秒处）必须100%还原上传海报——产品、文字、logo、颜色、版式，每一个元素都不能变
- 9:16竖版，适合抖音开屏广告

动画时间线：
- 0-1秒：只展示海报背景（原木色调餐厅场景），镜头做极缓慢推近+光线微微变亮。背景上没有任何产品、没有任何文字
- 1-2.4秒："山脊"餐桌从画面下方自然滑入到海报中的原始位置，滑入过程中带轻微运动模糊。产品外观（黑胡桃木色、锥形腿造型）保持与海报完全一致
- 2.4-3.3秒：主标题"618狂欢·全店7折"从画面中央放大出现，锁定到海报原始位置。字体样式与海报一致
- 3.3-4秒：副标题、品牌logo"树觉生活"、价格标签逐个以淡入方式到位
- 4-5秒：完整海报画面保持微小的上下呼吸运动（幅度极小，像一张有生命的海报）。不新增任何元素

禁止：
- 不要切换场景
- 不要新增粒子、光斑、花瓣、烟雾等任何装饰效果
- 不要改动任何文字内容
- 不要改动产品外观

风格：品牌广告级的动态海报——干净、克制、商务感
```

**测试关注点**：

| 关注项 | 描述 |
|--------|------|
| 最终帧还原度 | 第5秒画面与上传海报是否逐元素一致 |
| 动画节奏 | 分层进入时间是否合理，是否突兀 |
| 产品稳定性 | 餐桌在动画过程中是否变形/变色/模糊 |
| 文字保真 | 中文字是否全程清晰、无乱码 |
| 零幻觉遵守 | 是否出现了禁止的粒子、花瓣、光斑等元素 |

### 4.2 场景发散

- **品牌官网**：静态KV→首页动态开屏banner
- **线下门店**：促销POP→电子屏动态展示
- **展会/发布会**：Keynote首页→动态开场
- **电商详情页**：主图→动态首屏视频模块
- **电梯广告**：平面广告→动态轮播


## 📌 案例五：产品照片→安装说明书（售后物料）

### 造假背景

**品牌**："木壳MUKE"（虚构），简约家具品牌。淘宝/拼多多卖组装家具——书架、衣帽架、边几、书桌。产品不复杂（每个大概5-10个部件），但每款都需要配安装说明书。

**困境**：用CAD画安装说明图，一款产品平均耗时1.5天。月均上新3-5款=4.5-7.5天耗在说明书上。且客服每天还有40+条"这个怎么装"的重复咨询。招一个专职画说明图的设计师北京月薪10K+，但月销售额才30万。

**真实痛点**：组装家具的售后成本大量消耗在"教用户安装"上。一张清晰的安装说明书=减少客服压力+降低退货率（装不起来就退）。

### 5.1 场景测试反馈

**测试场景**：组装家具 → IKEA风格安装说明书

**突出用法（区别于普通模式）**：
- doubao-creative-design的infographic路由专长做"信息结构化视觉图"
- 交付物是可直接打印、直接放进包装盒的成品说明书——不需要设计师再做排版
- 从产品实物图逆向推导结构→编写步骤→生成指引图

**提示词**：

```
上传这款三层旋转书架的产品照片（组装完成状态）。

帮我生成一张IKEA风格的安装说明书图片。

产品：三层旋转书架
材质：橡木色MDF板+金属转轴
部件：顶板×1、底板×1、侧板×2、层板×3、背板×1、螺丝×12、L型角码×8、六角扳手×1

要求：
- 白色背景，IKEA风格的线条图风格
- 顶部标题："三层旋转书架·安装说明"（大号粗黑体）
- 左侧：零件清单区——每个零件画简笔图标+编号（A-H）+名称+数量（如"A·顶板·1块"）
- 右侧：6个安装步骤，圆圈编号+箭头连接：
  步骤1：将侧板与底板用角码固定（4颗螺丝）
  步骤2：安装背板，对齐卡槽后拧螺丝
  步骤3：从下往上依次安装三层隔板（注意间距一致）
  步骤4：安装顶板，对齐四角
  步骤5：检查所有螺丝是否拧紧，旋转结构是否顺畅
  步骤6：完成，放置于平整地面，检查整体稳定性
- 底部安全提示条："⚠ 单层承重≤15kg · 请勿儿童攀爬 · 定期检查螺丝是否松动"
- A4纵向比例
- 线条用深灰主线+蓝色结构线（箭头+编号用蓝色）
- 整体干净、专业、可直接打印放入包装盒
```

**测试关注点**：

| 关注项 | 描述 |
|--------|------|
| 结构逻辑 | 组装步骤是否逻辑正确（装背板前不能装隔板） |
| 零件完整性 | 零件清单是否包含所有部件 |
| 中文印刷质量 | 所有文字（标题+零件名+步骤说明+安全提示）是否清晰可读 |
| 打印适配 | A4比例+分辨率是否适合打印 |
| 简洁度 | 图是否太密或太乱，安装指引是否一目了然 |

### 5.2 场景发散

- **STEAM教具/积木**：产品图→小朋友能看懂的拼装图
- **健身器材**：哑铃凳→组装+姿势指引
- **家电**：空气炸锅→首次使用+清洗步骤说明
- **宠物用品**：猫爬架→安装+使用安全说明
- **儿童玩具**：滑板车→组装+安全须知+适用年龄


## 📌 案例六：门店实拍→15秒店铺宣传短视频

### 造假背景

**店铺**："叁分之一咖啡"（虚构），杭州西湖区一条巷子里的社区精品咖啡店。开了8个月，线下回头客还行，但线上几乎零存在。

**困境**：老板小米想在抖音和小红书推一下。请达人探店一条报价800-3000，且达人拍出来的风格和她店里真实的氛围完全不同——"我的店是温暖的、安静的，达人拍出来就是网红ins风"。自己拿手机拍又只有固定机位的长镜头，放抖音根本没人看。

**真实痛点**：线下店需要"低成本+高还原+有视频质感"的店铺宣传内容。传统路径（请达人 vs 自己拍）两头都不对。

### 6.1 场景测试反馈

**测试场景**：社区咖啡店 → 15秒店铺宣传短视频

**突出用法（区别于普通模式）**：
doubao-creative-video的Marketing路由（非UGC）在这种场景下更合适——因为要的不是"达人种草"的口播感，而是"展示这家店是什么样"的视觉体验。核心差异：
1. **基于实拍做视频化延展**：不是AI从零编一个咖啡店，而是基于门头/室内/吧台/饮品的实拍照片，生成一个能代表店铺真实氛围的短视频
2. **氛围还原优先**：视频的核心任务是让观众看完想"去这家店坐坐"——不是硬卖产品，而是传递空间的温度和情绪

**提示词**：

```
上传我家咖啡店的实拍照片（门头、室内全景、吧台特写、招牌饮品）。

帮我生成一条15秒的店铺宣传短视频，用于抖音和小红书。

店铺信息：
- 店名：叁分之一咖啡（1/3 Coffee）
- 风格：日系简约社区咖啡，温暖安静
- 特色：自家烘焙豆子、开放式手冲吧台、每周更换单品豆
- 招牌：桂花酒酿拿铁（有桂花干撒表面）、冰博客Dirty（牛奶和咖啡的分层）
- 氛围关键词：治愈、慢、一个人待一下午的那种舒服

视频要求：
- 9:16竖版，15秒
- 不是UGC口播推荐——是品牌营销短视频（Marketing路线）
- 镜头节奏：
  0-3秒：门头推近——从巷子口推进到玻璃门（木色门框+白色招牌）
  3-7秒：室内慢扫——手冲吧台区（咖啡师在注水）+靠窗双人座（阳光斜入+绿植）+角落书架
  7-11秒：饮品特写——桂花酒酿拿铁的桂花干特写+冰博客Dirty的奶咖分层+蒸汽上升
  11-15秒：门口外摆位+暖光吊灯，字幕"周末下午三点，这里总有位置等你"
- 风格：日系治愈Vlog质感、温暖慢节奏。不要快剪、不要炫酷转场
- 无旁白（纯BGM+字幕），BGM：舒缓Jazz/Acoustic Guitar
- 字幕：日系细体半透明底，白色字
- 产品特写中的咖啡外观与上传照片保持一致
```

**测试关注点**：

| 关注项 | 描述 |
|--------|------|
| 场景还原度 | 视频中的餐厅布局、装修风格是否基于上传实拍 |
| 氛围准确性 | 是否保持了日系治愈而不是变成网红ins风 |
| 产品特写保真 | 咖啡饮品的分层/桂花干等细节是否真实 |
| 节奏控制 | 15秒四段分配是否符合慢节奏店铺的调性 |
| 无虚假信息 | 不编造评分、不编造客户评价 |


## 📌 案例七：食品→白底商业广告大片（设计）

### 造假背景

**品牌**："唐点"（虚构），广式点心品牌。主打虾饺、流沙包、凤爪等传统点心，供应给商超和电商。创始人李先生的目标是让广式点心进入"高端速冻食品"赛道，对标湾仔码头。但产品包装可以花钱设计，产品照却一直凑合——用手机拍的白底图，虾饺皮的反光和馅料透出的粉色完全没出来。

**真实痛点**：食品电商的生死线是图片能不能让用户"看饿了"。传统食品摄影一个单品2000-5000元，对于SKU多的食品品牌来说，拍照成本占总营销成本的30%以上。

### 7.1 场景测试反馈

**测试场景**：速冻虾饺→白底广告级产品照

**提示词**：

```
帮我生成一张虾饺的白底广告级产品照。

产品信息：
- 品类：速冻广式虾饺
- 外观：半透明澄面皮，能隐约看到里面粉色的虾仁馅
- 蒸熟后：皮晶莹剔透、虾仁Q弹可见、饺边褶皱清晰
- 规格：一笼4只，竹蒸笼上

要求：
- 白底商业摄影风格
- 蒸笼位于画面偏上方1/3处，虾饺排列整齐
- 用筷子夹起一只虾饺，离蒸笼约3cm，虾饺皮在光线下晶莹剔透
- 馅料中的虾仁隐约可见淡粉色
- 旁边有极细微的蒸汽（不要太浓，刚好让人感觉是热的）
- 光线：左侧柔光箱打亮虾饺皮的高光区，让皮的透明感出来
- 整体风格：高端速冻食品包装级别的商业摄影，不是路边摊
- 比例：1:1正方形
- 不要加任何logo、文字、价格
```

### 7.2 场景发散

这个方向直接对应Dola Row 30（食品→白底广告大片），在国内的可衍生场景：
- 烘焙/甜品：抹茶千层、提拉米苏→白底甜品菜单级照片
- 肉类/海鲜：和牛切片、三文鱼刺身→高品质食材展示图
- 饮品：手冲咖啡、精酿啤酒→产品图+广告级场景图
- 调味料：手工辣酱、松露油→瓶身质感+烹饪场景

## 整体评估

### Dola→国内升级的核心差异

| 维度 | Dola（海外版） | 国内豆包升级策略 |
|------|---------------|-----------------|
| 电商平台 | Shopify/Amazon规范 | 淘宝/拼多多/抖音三平台规范（主图尺寸/白底/卖点层级各自不同） |
| 社交传播 | Instagram/TikTok | 小红书+朋友圈+抖音三件套（风格必须差异化，不能一套图全平台发） |
| 赚钱感知 | "Save time and money" | "省了多少钱、几天回本、找设计师报价vs AI成本"——必须可量化 |
| 中文文字 | 英文短句为主 | 中文大标题/促销标签——对字体渲染要求极高 |
| UGC带货 | 海外达人生态 | 国内达人生态更卷——去AI腔+真实感要求更高 |
| 合规 | 相对宽松 | 食品/保健品/美妆有广告法限制——不编功效、不P过狠 |

### 风险标注

| 风险 | 等级 | 描述 |
|------|------|------|
| 中文文字渲染 | 🔴 阻塞级 | 所有案例依赖中文文字准确渲染，如果文字乱码则案例不可用 |
| 产品一致性锁（多图） | 🔴 阻塞级 | I2I链路是否正确（而非每次独立文生图） |
| UGC脚本去AI腔 | 🟡 高风险 | 口播是否像真人而非AI念稿 |
| 光影融合自然度 | 🟡 中风险 | 产品在新场景中是否贴合 |
| 主播去重 | 🟢 低风险 | 批量视频间主播是否撞脸 |
