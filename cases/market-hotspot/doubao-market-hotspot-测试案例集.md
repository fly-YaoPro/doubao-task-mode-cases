# doubao-market-hotspot Skill 测试案例集

> 生成时间：2026-06-29
> Skill本质：面向普通股民的市场整体与宏观事件解读——对话框深度正文字交付，HTML为可选项
> 与 doubao-finance-sector 的本质区别：sector看板块热度（双轨打分+10代表股），market-hotspot看市场整体影响（7段叙事+事件传播链）
> 7类问题：全市场热点 / 涨跌归因 / 宏观政策 / 新闻风险 / 跨资产联动 / 风险偏好情绪 / 市场级更新追踪

## 案例一：今天市场为什么跌？（涨跌归因——核心高频场景）

### 一、案例设计

| 维度 | 内容 |
|------|------|
| **触发类型** | 类型2：市场涨跌归因（"今天A股为什么下跌？"） |
| **默认时间窗** | 当日交易时段，盘前看隔夜外盘+今日预期 |
| **7段叙事** | 核心观点→有哪些关键增量信息→为什么重要→如何影响市场→各方观点→验证/证伪信号→信息来源 |

### 二、测试提示词

```
今天A股为什么跌这么多？帮我看看是什么原因。
```

### 三、预期Skill路由

```
用户prompt → 类型2：市场涨跌归因
↓
判定时间窗：当前交易时段 → 覆盖当日已发生+隔夜外盘
↓
读取 question-types.md → dialog-delivery.md → search-playbook.md
↓
搜索：指数表现/风格切换/成交量/涨跌家数/领跌方向/重要消息
  - 搜索"A股 今日 下跌 原因"
  - 搜索"今日A股 领跌板块"
  - 搜索"隔夜美股 外盘"（如果盘前）
  - 搜索最新政策/事件公告
↓
去重合并：同事件多源合并，保留最权威来源
↓
按7段结构写正文章稿
↓
自检：栏目标题=HTML同名、有关键事实来源、无推荐建议、结尾固定提示
```

### 四、测试关注点

| # | 关注点 | 通过标准 | 常见错误 |
|---|--------|---------|---------|
| 1 | **触发识别** | "为什么跌"触发skill，"某只股票为什么跌"转个别股日报 | 单股问题也触发了skill |
| 2 | **7段结构完整** | 核心观点→增量信息→为什么重要→如何影响→各方观点→验证信号→来源 | 缺少某段或用自拟标题替代 |
| 3 | **不输出表格** | 用分组bullet而非Markdown表格 | 用表格展示涨跌板块 |
| 4 | **来源可点击** | 关键事实配有[1][2]引用编号，来源区有URL | 只写"据媒体报道"无链接 |
| 5 | **不推荐股票** | 全文无"推荐""建议买入""目标价" | 在"如何影响"段里推了受益板块个股 |
| 6 | **结尾固定提示** | 最后一行逐字输出"需要为你生成一份精美的热点报告么？" | 改写措辞或跳过 |


## 案例二：央行降息对股市有什么影响？（宏观政策——传播链）

### 一、案例设计

| 维度 | 内容 |
|------|------|
| **触发类型** | 类型3：宏观/政策事件影响（"降息""利率决策""央行政策"） |
| **7段侧重** | 政策原文→权威解读→影响变量（利率→估值/资金成本→行业）→受影响的资产环→市场预期差→后续验证 |

### 二、测试提示词

```
最近央行又降息了，这对A股有什么影响？帮我梳理一下降息是怎么传导到股市的。
```

### 三、测试关注点（补充案例一未覆盖的）

| 关注点 | 通过标准 | 风险点 |
|--------|---------|--------|
| **政策原文引用** | 引用央行公告原文或权威财经源 | 用自媒体解读代替官方源 |
| **传导链拆解** | 事实→变量→资产/市场反应→后续验证，逐环写清 | 直接跳到结论"利好股市" |
| **预期差** | 说明"市场之前预期什么、实际是什么、差在哪" | 只讲政策内容不讲预期 |
| **反向证据** | 如果降息已被定价（市场不涨反跌），需要解释为什么 | 只讲利好不讲利空 |


## 案例三：欧洲央行加息，美联储会跟吗？（跨央行政策联动+概率推演）

### 一、案例设计

| 维度 | 内容 |
|------|------|
| **触发类型** | 类型3+5：宏观政策+跨资产联动（"ECB加息""Fed会跟吗"） |
| **特殊处理** | "会不会"类问题→条件推演而非预测——把判断写成"哪些事实支持跟、哪些事实支持不跟、什么信号会确认" |

### 二、测试提示词

```
欧洲央行6月又加息了，美联储后面会不会也跟着加？
这对美股和港股有什么影响？
```

### 三、测试关注点

| 关注点 | 通过标准 |
|--------|---------|
| **条件推演** | "会不会"→列出支持/反对的条件+关键验证信号，不是给一个概率数字 |
| **跨资产传导** | ECB加息→欧元/美元→新兴市场资金流→港股/A股，链路清楚 |
| **不以"预测"包装** | 不写"我们认为70%概率会加息"——只写条件推演 |


## 案例四：最近市场热点和交易主线（全市场热点扫描）

### 一、案例设计

| 维度 | 内容 |
|------|------|
| **触发类型** | 类型1：全市场热点/交易主线 |
| **7段侧重** | 5-8条热点→风格切换→成交量能→资金流向→情绪 |

### 二、测试提示词

```
最近A股市场的主要热点和交易主线有哪些？帮我梳理一下。
```

### 三、测试关注点

| 关注点 | 通过标准 | 风险点 |
|--------|---------|--------|
| **热点数量** | 5-8条，每条有事实+市场反应+来源 | 只列了3条或堆了15条无重点 |
| **风格判断** | 说明当前是成长/价值/小盘/大盘哪种风格占优 | 只列热点没说风格 |
| **不越界到板块** | 说"科技板块活跃"但不做板块级深度分析（那是finance-sector干的） | 开始分析板块热度 |


## 案例五：原油暴涨对全球股市影响（跨资产联动）

### 二、测试提示词

国际原油价格突然暴涨了15%，这对全球股市和通货膨胀预期有什么影响？

## 案例六：市场风险偏好是不是变弱了？（情绪/资金面）

### 二、测试提示词

```
最近感觉A股市场情绪不太好，资金风险偏好是不是在变弱？帮我看看。
```

### 三、测试关注点：是否覆盖了成交/波动/涨跌停/行业扩散/资金流/避险资产这六个情绪维度


## 与 doubao-finance-sector 的对比

| 维度 | finance-sector | market-hotspot |
|------|---------------|----------------|
| 分析粒度 | 板块/概念级 | 市场/宏观级 |
| 核心方法 | 双轨打分（信息+行情）×10代表股 | 7段叙事+事件传播链 |
| 数据依赖 | seed_finance_search强制 | 公开信息搜索 |
| 脚本流水线 | 有（derive→lint→repair→payload→render） | 默认无（dialog-only），HTML走外部模型路径 |
| 默认输出 | 对话框正文+固定结尾"生成HTML" | 对话框正文+固定结尾"需要生成热点报告么？" |
| 不应触发 | 宏观政策/大盘/单股 | 板块概念/个股/行业/公司 |
